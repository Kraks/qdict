qdict
=====

version 0.02

A simple network English-Chinese dictionary and named from "quick dict".

Need network connection and then qdict get xml data file from Youdao webdict service.

Thanks for libcurl and mini-xml.

Usage
-----

	./qdict [word]

	interactive mode:
		./qdict -i
