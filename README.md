qdict
=====

version 0.1

A fast, simple English-Chinese dictionary.

Need network and Youdao dictionary web service for query, and then will save the translation to local db.

Use Berkeley DB, libcurl and mini-XML library.

Usage
-----

qdict [word] [+]  
-i, --interactive  
-w, --wordbook  
-h, --help
